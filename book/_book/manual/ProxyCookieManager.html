水星会话凭证管理器

水星会话管理器

用于管理 cookie 的类。除非另有说明, 否则可以在任何线程上调用此类的方法。

功能函数

是否有效
此函数检查本类中的对象实例是否有效。
@return 逻辑型
取所有会话凭证
访问所有cookies。 返回的cookie由事件\"OnCookieVisitor\"传递, cookie的传递顺序根据最长路径来选择, 在然后是最早的创建日期。
@return 水星会话凭证
取指定会话凭证
访问指定url下的所有cookies。返回的cookie由事件\"OnCookieVisitor\"传递, cookie的传递顺序根据最长路径来选择, 在然后是最早的创建日期。
@param 文本型 url url
@param 逻辑型 includeHttpOnly includeHttpOnly
@return 水星会话凭证
置会话凭证
设置一个具有有效URL和cookie属性的cookie。设置的cookie属性应该使用标准的格式输入, 在此函数中将会检查cookie中是否存在非法字符(例如: \";\"是不允许出现在cookie.value中的),如果出现了这样的字符此函数将不会设置此cookie属性并返回失败。
@param 文本型 url url
@param 文本型 domain domain
@param 文本型 name name
@param 文本型 value value
@return 逻辑型
删除会话凭证
删除与指定参数匹配的cookie。如果指定了 url 和 cookie_name 那么将仅删除与其相符的cookie。如果仅指定 url 所有与其相符的cookie都将删除(但不包括域cookie)。如果 url 为空则将删除所有主机和域的cookies。
@param 文本型 url url
@param 文本型 凭证名称 凭证名称
@return 逻辑型
刷新到存储
将在内存中的存储数据(如果有的话)写入到磁盘上。
@return 逻辑型
保存到文本
将当前cookie管理器的所有数据保存到文本字符串。
@return 文本型
从文本还原
从文本字符串中还原所有cookie数据到管理器。
@param 文本型 cookie数据 cookie数据
@return 无返回值
保存为HTTP格式
将当前cookie管理器中的指定数据以HTTP会话请求所用的格式保存到文本字符串。
@param 文本型 url url
@return 文本型
